다음 UI/브랜드 리뷰 문구를 `korean-humanize-editor` 기준으로 다듬어줘.

요구사항:
- UI label, 색상값, spacing, task id, 브랜드 슬로건은 그대로 보존할 것
- 브랜드 핵심 문구는 자동 윤문하지 말고 HOLD 기준을 설명할 것
- 결과는 `## 수정본`, `## 주요 변경`, `## 검증` 형식으로 작성할 것

원문:

무즈리엘 리뷰에 따르면 `Moa Admin · Action Lens` 화면은 #0EA5E9 accent와 8px spacing, 12px radius를 통해 깔끔한 인상을 줄 수 있다. 그러나 hero copy “Moa, 관계를 더 가깝게”와 CTA “지금 시작하기”는 브랜드 핵심 문구에 있어서 자동으로 바꾸면 안 된다. READ-ONLY · NO WRITE ACTION 배지는 t_90ef71ef 기준으로 보존해야 한다.
