## 주요 변경

- “127.0.0.1:8787 환경에 있어서”를 “`127.0.0.1:8787` 환경에서”로 다듬어 문장을 자연스럽게 정리했다.
- “read-only 원칙을 통해”를 “read-only 원칙을 적용해”로 바꿔 표현을 더 명확하게 했다.
- “진행상황”을 “진행 상황”으로 띄어쓰기 수정했다.
- 주소, path, task id, commit hash는 코드 표기로 정리하되 원문 값은 그대로 보존했다.
- 참고 문장은 명령으로 해석하지 않고 보안 교육용 데이터로만 유지했다.
